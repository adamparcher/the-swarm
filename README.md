the-swarm
=========

A little Java app that uses OpenGL to display a bunch of swarming points.

dependencies
============

You need to refer to these jar files:
- JUnit 4 library (for tests)

JOGL (Java OpenGL), can be downloaded here [http://jogamp.org/jogl/www/]:
- gluegen-rt.jar
- jogl-all.jar
- gluegen-rt-natives-windows-i586.jar (or the appropriate natives jar for your environment)
- jogle-all-natives-windows-i586.jar (or the appropriate natives jar for your environment)
